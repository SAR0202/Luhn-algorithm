# Luhn-algorithm/Algortimo de Luhn
The Luhn algorithm is used to verify the validity of a credit card.In this project I created a simple function to detect whether a given  credit card number is valid or not, and to which company it belongs.

El algoritmo de Luhn se utiliza para verificar la validez de una tarjeta de crédito. En este proyecto creé una función simple para detectar si un número de tarjeta de crédito dado es válido o no, y a qué empresa pertenece.
